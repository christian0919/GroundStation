### HADWARE REQUIREMENTS

This ground station needs a **RYLR998** LoRa module connected via a **TTL to USB converter**

Pin connection of the TTL adapter and the RYLR998 module:
| TTL TO USB | RYLR998 |
|------------|---------|
|     TX     |   RX    |
|     RX     |   TX    |
|     GND    |   GND   |
|     3V3    |   VDD   |
